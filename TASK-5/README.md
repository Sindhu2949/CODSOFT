# Handwritten Text Generation using RNN (LSTM)

## Project Overview
This project implements a Character-Level Recurrent Neural Network (RNN) using LSTM to generate handwritten-like text patterns. The model learns patterns from a dataset of text and generates new text based on the learned structure.

The objective is to simulate handwritten-style text generation using deep learning techniques.

This project is developed as part of an AI/ML Internship task.

---

## Technologies Used

- Python
- TensorFlow
- NumPy
- Deep Learning (RNN / LSTM)

---

## Dataset

The dataset used for training contains large text samples which simulate handwritten-style text patterns.

Dataset Source: Hugging Face
