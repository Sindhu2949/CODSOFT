# 🤖 CodSoft Machine Learning Internship Projects

This repository contains the projects completed during the **CodSoft Machine Learning Internship Program**.
The goal of these tasks is to implement various **Machine Learning and Deep Learning models** to solve real-world problems.

# 📂 Internship Tasks

## 🎬 Task 1: Movie Genre Classification

### 📌 Objective

Build a machine learning model that predicts the **genre of a movie** based on its **plot summary or textual description**.

### 🛠 Techniques Used

* Natural Language Processing (NLP)
* TF-IDF Vectorization
* Logistic Regression

### 📊 Workflow

1. Load and clean movie dataset
2. Convert text into numerical features using **TF-IDF**
3. Train a **Logistic Regression classifier**
4. Evaluate model performance using accuracy and classification metrics

# 💳 Task 2: Credit Card Fraud Detection

### 📌 Objective

Develop a machine learning model to detect **fraudulent credit card transactions**.

### 🛠 Algorithms Used

* Logistic Regression
* Random Forest
* Decision Trees

### 📊 Workflow

1. Load credit card transaction dataset
2. Handle class imbalance
3. Train classification model
4. Predict whether a transaction is **fraudulent or legitimate**

# 📉 Task 3: Customer Churn Prediction

### 📌 Objective

Predict whether a **customer will leave a subscription-based service**.

### 🛠 Algorithms Used

* Logistic Regression
* Random Forest
* Gradient Boosting

### 📊 Workflow

1. Data preprocessing
2. Feature selection
3. Model training
4. Customer churn prediction

# 📱 Task 4: Spam SMS Detection

### 📌 Objective

Create an AI model that classifies **SMS messages as Spam or Legitimate**.

### 🛠 Techniques Used

* TF-IDF Vectorization
* Naive Bayes Classifier
* Logistic Regression

### 📊 Workflow

1. Load SMS dataset
2. Text preprocessing
3. Convert text to TF-IDF features
4. Train spam classifier

# ✍️ Task 5: Handwritten Text Generation

### 📌 Objective

Implement a **Character-Level Recurrent Neural Network (RNN)** to generate handwritten-style text.

### 🛠 Technologies Used

* TensorFlow
* Keras
* LSTM (Recurrent Neural Networks)

### 📊 Workflow

1. Load text dataset
2. Create character-level vocabulary
3. Train LSTM model
4. Generate new text based on learned patterns
