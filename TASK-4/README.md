# Spam SMS Detection

## 📌 Objective
To develop a Machine Learning model that classifies SMS messages as Spam or Ham (Legitimate).

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Naive Bayes Classifier

## 📊 Model Used
Multinomial Naive Bayes

## 📈 Results
Achieved high accuracy in detecting spam messages using TF-IDF features.

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python spam_detection.py

## 📚 Dataset
SMS Spam Collection Dataset
