# Customer Churn Prediction

## Objective
Predict customer churn for a subscription-based service.

## Algorithms Used
- Logistic Regression
- Random Forest
- Gradient Boosting

## Techniques Used
- Data Cleaning
- Feature Encoding
- Feature Scaling
- Model Evaluation using ROC-AUC

## How to Run
pip install -r requirements.txt
python churn_prediction.py

## Output
- Classification Report
- Confusion Matrix
- ROC Curve
- Saved Best Model
